# v4.7 PRO Sub-Spec Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the four PRO items from the v4.7 PRO sub-spec (waste tracker, reposition prediction, PRO trial 7d, pricing visible in Settings) so PRO conversion can move from 0% to 1–3% and waste/reposition insights become reachable behind a 7-day trial.

**Architecture:** All compute lives in pure domain functions under `core/domain/insights/`. Page-scoped `InsightsStateService` exposes `wasteSummary` and `repositionPredictions` computed signals derived from the existing `events` and `pantryStore.items()` sources. Dashboard consumes the same state via two new standalone components; Insights tab gains two new sections (above existing AI insights). The 7-day free trial is configured in the RevenueCat dashboard — the client surfaces a reusable `ProTrialCtaComponent` across `/upgrade`, Settings PRO hero, Insights free state, the waste card free state and the reposition card free state. Trial-start and trial-expiry events fire from a localStorage-backed detector in `AppComponent` on resume. Per-session CTA dismissal is in-memory (not persisted).

**Tech Stack:** Angular 20 standalone components, Signals + computed, Ionic 8, RxJS only where it already exists (RevenueCat service), `@revenuecat/purchases-capacitor` v11.2.15, PostHog via the existing `AnalyticsService`, PouchDB (read-only here — events already persisted).

```ts
private readonly trialEligibleSubject = new BehaviorSubject<boolean>(false);
readonly hasUnusedTrial$: Observable<boolean> = this.trialEligibleSubject.asObservable();

/** Recomputes trial eligibility from the current offering's intro price. */
async refreshTrialEligibility(): Promise<void> {
  const offering = await this.getOfferings();
  if (!offering) {
    this.trialEligibleSubject.next(false);
    return;
  }
  const monthly = offering.monthly;
  const annual = offering.annual;
  const monthlyTrial = Boolean(monthly?.product?.introPrice?.price === 0);
  const annualTrial = Boolean(annual?.product?.introPrice?.price === 0);
  // If PRO is already active, the user is not eligible regardless of intro price.
  if (this.isPro()) {
    this.trialEligibleSubject.next(false);
    return;
  }
  this.trialEligibleSubject.next(monthlyTrial || annualTrial);
}
```

Then call `await this.refreshTrialEligibility();` at the end of the existing `init(userId)` method (just before the closing `} catch`).

Defensive: any throw → `hasUnusedTrial$` stays `false`. Falling back to the plain upgrade button is always safe.

```ts
import { Injectable, computed, signal } from '@angular/core';

export type ProCtaSurface =
  | 'upgrade_page'
  | 'settings_hero'
  | 'insights_tab'
  | 'waste_card'
  | 'reposition_card';

@Injectable({ providedIn: 'root' })
export class ProCtaUiStateService {
  private readonly dismissed = signal<ReadonlySet<ProCtaSurface>>(new Set());

  isDismissed(surface: ProCtaSurface): boolean {
    return this.dismissed().has(surface);
  }

  dismiss(surface: ProCtaSurface): void {
    const next = new Set(this.dismissed());
    next.add(surface);
    this.dismissed.set(next);
  }

  readonly dismissedSurfaces = computed(() => this.dismissed());
}
```

```ts
import {
  ChangeDetectionStrategy, Component, Input, inject, signal, effect,
} from '@angular/core';
import { CommonModule } from '@angular/common';
import { IonicModule } from '@ionic/angular';
import { Router } from '@angular/router';
import { TranslateModule } from '@ngx-translate/core';
import { UpgradeRevenuecatService } from '@core/services/upgrade/upgrade-revenuecat.service';
import { AnalyticsService } from '@core/services/analytics/analytics.service';
import { ProCtaUiStateService, type ProCtaSurface } from '@core/services/upgrade/pro-cta-ui-state.service';
import { STORAGE_KEYS } from '@core/constants';

@Component({
  selector: 'app-pro-trial-cta',
  standalone: true,
  imports: [CommonModule, IonicModule, TranslateModule],
  changeDetection: ChangeDetectionStrategy.OnPush,
  templateUrl: './pro-trial-cta.component.html',
  styleUrl: './pro-trial-cta.component.scss',
})
export class ProTrialCtaComponent {
  private readonly revenueCat = inject(UpgradeRevenuecatService);
  private readonly analytics = inject(AnalyticsService);
  private readonly router = inject(Router);
  private readonly ctaUi = inject(ProCtaUiStateService);

  @Input({ required: true }) surface!: ProCtaSurface;
  /** When true, render a smaller variant suitable for inline placement. */
  @Input() compact = false;
  /** When true, allow the user to dismiss the CTA for this session. */
  @Input() dismissible = false;

  readonly hasUnusedTrial = signal<boolean>(false);
  readonly busy = signal<boolean>(false);

  constructor() {
    this.revenueCat.hasUnusedTrial$.subscribe(v => this.hasUnusedTrial.set(v));
  }

  get hidden(): boolean {
    return this.ctaUi.isDismissed(this.surface);
  }

  async onPrimary(): Promise<void> {
    if (this.busy()) return;
    this.busy.set(true);
    this.analytics.trackProTrialCtaClicked(this.surface);
    try {
      if (this.hasUnusedTrial()) {
        const ok = await this.revenueCat.purchasePro();
        if (ok) {
          localStorage.setItem(STORAGE_KEYS.PRO_TRIAL_STARTED_AT, new Date().toISOString());
          this.analytics.trackProTrialStarted();
        }
      } else {
        this.router.navigate(['/upgrade']);
      }
    } finally {
      this.busy.set(false);
    }
  }

  onDismiss(): void {
    this.ctaUi.dismiss(this.surface);
  }
}
```

```ts
import type { PantryEvent } from '@core/models/events';
import type { FoodType } from '@core/models/shared/enums.model';

export interface WasteSummary {
  windowDays: number;
  totalCount: number;
  byCategory: Array<{ categoryId: string; count: number }>;
  byFoodType: Array<{ foodType: FoodType; count: number }>;
  topProduct?: { productId: string; productName: string; count: number };
  previousWindowCount: number;
  trend: 'up' | 'down' | 'flat';
}

const MS_PER_DAY = 86_400_000;

export function computeWasteSummary(
  events: ReadonlyArray<PantryEvent>,
  now: Date,
  windowDays: number,
): WasteSummary {
  const nowMs = now.getTime();
  const windowStart = nowMs - windowDays * MS_PER_DAY;
  const prevStart = windowStart - windowDays * MS_PER_DAY;

  const inWindow: PantryEvent[] = [];
  let previousWindowCount = 0;

  for (const e of events) {
    if (e.eventType !== 'EXPIRE') continue;
    const t = new Date(e.timestamp).getTime();
    if (Number.isNaN(t)) continue;
    const q = Number.isFinite(e.quantity) ? e.quantity : 0;
    if (t >= windowStart && t <= nowMs) {
      inWindow.push(e);
    } else if (t >= prevStart && t < windowStart) {
      previousWindowCount += q;
    }
  }

  const byCategoryMap = new Map<string, number>();
  const byFoodTypeMap = new Map<FoodType, number>();
  const byProductMap = new Map<string, { productName: string; count: number }>();
  let totalCount = 0;

  for (const e of inWindow) {
    const q = e.quantity;
    totalCount += q;
    if (e.categoryId) byCategoryMap.set(e.categoryId, (byCategoryMap.get(e.categoryId) ?? 0) + q);
    if (e.foodType)   byFoodTypeMap.set(e.foodType, (byFoodTypeMap.get(e.foodType) ?? 0) + q);
    const productName = e.productName ?? '';
    const existing = byProductMap.get(e.productId);
    if (existing) {
      existing.count += q;
    } else {
      byProductMap.set(e.productId, { productName, count: q });
    }
  }

  const byCategory = [...byCategoryMap.entries()]
    .map(([categoryId, count]) => ({ categoryId, count }))
    .sort((a, b) => b.count - a.count);
  const byFoodType = [...byFoodTypeMap.entries()]
    .map(([foodType, count]) => ({ foodType, count }))
    .sort((a, b) => b.count - a.count);

  let topProduct: WasteSummary['topProduct'];
  let top = 0;
  for (const [productId, v] of byProductMap) {
    if (v.count > top) {
      top = v.count;
      topProduct = { productId, productName: v.productName, count: v.count };
    }
  }

  let trend: WasteSummary['trend'] = 'flat';
  if (totalCount > previousWindowCount) trend = 'up';
  else if (totalCount < previousWindowCount) trend = 'down';

  return {
    windowDays,
    totalCount,
    byCategory,
    byFoodType,
    topProduct,
    previousWindowCount,
    trend,
  };
}
```

```ts
import type { PantryItem } from '@core/models/pantry/item.model';
import type { PantryEvent } from '@core/models/events';
import { sumQuantities } from '@core/domain/pantry/pantry-batch.domain';

export interface RepositionPrediction {
  productId: string;
  productName: string;
  categoryId?: string;
  currentStock: number;
  velocityPerDay: number;
  daysToOut: number;
  confidence: 'low' | 'medium' | 'high';
}

const WINDOW_DAYS = 30;
const MS_PER_DAY = 86_400_000;
const VELOCITY_THRESHOLD = 0.05;
const DAYS_TO_OUT_CAP = 90;

export function computeRepositionPredictions(
  items: ReadonlyArray<PantryItem>,
  events: ReadonlyArray<PantryEvent>,
  now: Date,
): RepositionPrediction[] {
  const nowMs = now.getTime();
  const windowStart = nowMs - WINDOW_DAYS * MS_PER_DAY;

  // Aggregate CONSUME quantities per product in window.
  const consumeAgg = new Map<string, { qty: number; count: number }>();
  for (const e of events) {
    if (e.eventType !== 'CONSUME') continue;
    const t = new Date(e.timestamp).getTime();
    if (Number.isNaN(t) || t < windowStart || t > nowMs) continue;
    const prev = consumeAgg.get(e.productId) ?? { qty: 0, count: 0 };
    prev.qty += Number.isFinite(e.quantity) ? e.quantity : 0;
    prev.count += 1;
    consumeAgg.set(e.productId, prev);
  }

  const out: RepositionPrediction[] = [];
  for (const item of items) {
    if ((item as any).isFresh) continue;
    const agg = consumeAgg.get(item._id);
    if (!agg) continue;
    const velocityPerDay = agg.qty / WINDOW_DAYS;
    if (velocityPerDay < VELOCITY_THRESHOLD) continue;
    const currentStock = sumQuantities(item.batches);
    if (currentStock <= 0) continue;
    const daysToOut = Math.min(DAYS_TO_OUT_CAP, Math.round(currentStock / velocityPerDay));
    const confidence: RepositionPrediction['confidence'] =
      agg.count >= 10 ? 'high' : agg.count >= 3 ? 'medium' : 'low';
    if (confidence === 'low') continue; // hide low-confidence
    out.push({
      productId: item._id,
      productName: item.name,
      categoryId: item.categoryId,
      currentStock,
      velocityPerDay,
      daysToOut,
      confidence,
    });
  }
  return out.sort((a, b) => a.daysToOut - b.daysToOut);
}
```

### Task E1: Detect trial expiry in `AppComponent` on resume

**Files:**
- Modify: `src/app/app.component.ts`

**Why:** When the trial expires, RevenueCat flips `isPro` from true to false. The client emits `pro_trial_expired` once per trial.

- [ ] **Step 1: Branch**

```bash
git checkout release/4.7 && git pull
git checkout -b feat/v4.7-pro-trial-expiry
```

- [ ] **Step 2: Add the detector**

In `AppComponent`'s init (or wherever `App.addListener('appStateChange', ...)` already lives), add:

```ts
import { STORAGE_KEYS } from '@core/constants';

private wasProLastCheck: boolean | null = null;

private detectTrialExpiry(): void {
  const isPro = this.revenueCat.isPro();
  const trialStartedAt = localStorage.getItem(STORAGE_KEYS.PRO_TRIAL_STARTED_AT);
  const fired = localStorage.getItem(STORAGE_KEYS.PRO_TRIAL_EXPIRED_FIRED);

  if (!trialStartedAt) {
    this.wasProLastCheck = isPro;
    return;
  }
  if (fired) {
    this.wasProLastCheck = isPro;
    return;
  }
  if (this.wasProLastCheck === true && isPro === false) {
    this.analytics.trackProTrialExpired();
    localStorage.setItem(STORAGE_KEYS.PRO_TRIAL_EXPIRED_FIRED, new Date().toISOString());
  }
  this.wasProLastCheck = isPro;
}
```

Call `detectTrialExpiry()` from:
- `ngOnInit` (or constructor) — initial bootstrap.
- The existing `App.addListener('appStateChange', state => { ... })` handler when `state.isActive === true`.

Make sure to also subscribe to `revenueCat.isPro$` so that a customerInfo update mid-session triggers the check:

```ts
this.revenueCat.isPro$.subscribe(() => this.detectTrialExpiry());
```
